👨‍💻 Personal Portfolio Website
A minimalist, high-performance personal portfolio website built with HTML5, CSS3, and Vanilla JavaScript. This project demonstrates modern web capabilities including a circular reveal dark mode theme toggle and real-time data visualization.

✨ Key Features
🌗 Circular Reveal Theme Toggle: A unique Dark/Light mode switch using the View Transition API to create a circular clip-path animation expanding from the cursor position.

⏳ Live Age Counter: A precise, real-time counter displaying age down to the 9th decimal place, updated every frame using requestAnimationFrame.

🇮🇳 Local Time Display: Dynamically calculates and displays the current time in the Asia/Kolkata (IST) timezone.

🖼️ 3D Tilt Effect: Interactive avatar container that tilts based on mouse movement coordinates using CSS transforms.

🎨 Modern UI/UX:

Glassmorphism elements and subtle SVG noise texture overlays.

Smooth page entry animations via CSS keyframes.

Fully responsive layout utilizing CSS Grid and Flexbox.

👁️ Visit Counter: A simulated view counter using localStorage with a smooth ease-out counting animation.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Variables, Flexbox, Grid), Vanilla JavaScript (ES6+).

Icons: FontAwesome 6 & Devicon.

Fonts: 'Inter' and 'Space Grotesk' via Google Fonts.

APIs: View Transition API, LocalStorage API, RequestAnimationFrame.

📂 Project Structure
/
├── index.html      # Main markup and content
├── style.css       # Styling, CSS variables, and keyframe animations
├── script.js       # Logic for age calculation, theme toggle, and tilt effects
└── logo.jpg        # Profile asset


Note: For the best experience with the View Transition API (Theme Toggle), use a Chromium-based browser (Chrome, Edge, Brave).
